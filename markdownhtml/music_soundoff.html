Music.SoundOff

Syntax

Music.SoundOff

Description

The Music.SoundOff procedure stops any sound or music that is currently playing or is waiting to play.

Status

Exported qualified. This means that you can only call the function by calling Music.SoundOff, not by calling SoundOff.
