precedence

Description

Turing's precedence rules determine the order of applying operators in an expression such as 3 + 4 * 5. These rules state, for example, that multiplication is done before addition, so this expression is equivalent to 3+ (4 * 5). Parenthesized parts of an expression are evaluated before being used. For example, in (1 + 2) * 3, the addition is done before the multiplication. The precedence rules are defined by this table, in which operators appearing earlier in the table are applied first.
