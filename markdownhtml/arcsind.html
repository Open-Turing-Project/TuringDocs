arcsind

Syntax

arcsind (r : real) : real

Description

The arcsind function is used to find the arc sine of an angle given in degrees. For example, arcsind (0.5) is 30.

Example

This program prints out the arcsine of values from -1 to 1 in degrees.

    for i : -4 .. 4
        const arg := i / 4
        put "Arc sine of ", arg, " is ",
            arcsind (arg), " degrees"
    end for
