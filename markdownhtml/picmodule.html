Pic

Description

This unit contains the predefined subprograms that deal with taking pictures of part of the screen, displaying them and moving pictures from file to screen and back. All routines in the Pic unit are exported qualified (and thus must be prefaced with "Pic.").

Entry Points

pic_new.htmlCreates a picture from a specified portion of the screen.pic_draw.htmlDraws a picture on the screen.pic_drawspecial.htmlDraws a picture on the screen using special effects.pic_drawspecialback.htmlDraws a picture on the screen while continuing execution using special effects.pic_free.htmlFrees up the picture created by using New or FileNew.pic_filenew.htmlCreates a picture from a file in an allowed format.pic_save.htmlSaves a picture as a file in an allowed format.pic_screenload.htmlDisplays a file in an allowed format on the screen directly.pic_screensave.htmlSaves a specified portion of the screen as a file in an allowed format.pic_rotate.htmlCreates a new picture by rotating an existing picture.pic_scale.htmlCreates a new picture by scaling an existing picture.pic_flip.htmlCreates a new picture by flipping an existing picture upside-down.pic_mirror.htmlCreates a new mirror-image of an existing picture.pic_blend.htmlCreates a new picture by blending two existing pictures together.pic_blur.htmlCreates a new picture by blurring an existing pictures.pic_width.htmlReturns the width of a picture.pic_height.htmlReturns the height of a picture.pic_frames.htmlReturns the the number of frames in a multi-frame GIF file.pic_filenewframes.htmlCreates an array of pictures from a multi-frame GIF file.pic_drawframes.htmlDraws a series of pictures on the screen in sequence while continuing execution.pic_drawframesback.htmlDraws a series of pictures on the screen in sequence.pic_settransparentcolor.htmlSets the color to be ignored when using picMerge mode.pic_settransparentcolor.htmlSets the color to be ignored when using picMerge mode.
