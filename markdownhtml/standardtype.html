standardType

Syntax

A standardType is one of: (a) int (b) real (c) string [ ( maximumLength ) ] (d) boolean (e) nat % natural number (f) intn % n-byte integer (n=1, 2, 4) (g) natn % n-byte natural (n= 1, 2, 4) (h) realn % n-byte real (n=4, 8) (i) char % single character (j) char(n) % n characters

Description

The standard types can be used throughout a program. They should not be included in an import list.

See also

int, real, string and boolean. See also nat, intn, natn, realn, char, char(n)
