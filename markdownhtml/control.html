Control Constructs

FOR

... statements ... exit when expn ... statements ...

LOOP

... statements ... exit when expn ... statements ...

IF

... statements ... { elsif condition then ... statements ... } [ else ... statements ... ]

CASE

... statements ... { label expn {, expn ... statements ... } [ label : ... statements ... ]

Any number of exit and exit when statements can appear at any place inside for .. end for constructs and loop .. end loop constructs.
