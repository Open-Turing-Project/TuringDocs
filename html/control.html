Control Constructs

FOR

 
for [ decreasing ] variable : startValue .. endValue
... statements ...
exit when expn
... statements ...
end for
 

LOOP

 
loop
... statements ...
exit when expn
... statements ...
end loop
 

IF

 
if condition then
... statements ...
{ elsif condition then
... statements ... }
[ else
... statements ... ]
end if
 

CASE

 
case expn of
... statements ...
{ label expn {, expn
... statements ... }
[ label :
... statements ... ]
end case
 

Any number of exit and exit when statements can appear at any place inside for .. end for constructs and loop .. end loop constructs.
