Config

Description   This unit contains the predefined subprograms that deal with getting configuration information about the machine and environment on which the program is being run. It exists in order to allow users to obtain information about the system that may only be available at run time.

All routines in the Config module are exported qualified (and thus must be prefaced with "Config.").

Entry Points  
Display Returns information about the display currently attached.
Lang Returns information about the language environment that the program is currently running within.
Machine Returns information about the hardware on which the program is running.
