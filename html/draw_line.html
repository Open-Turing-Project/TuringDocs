Draw.LinePart of Draw module

Syntax   Draw.Line (x1, y1, x2, y2, Color : int)

Description   The Draw.Line procedure is used to draw a line on the screen from (x1, y1) to (x2, y2) using the specified Color.

Example   This program draws a large X, reaching to each corner of the screen using color number 1. The maxx and maxy functions are used to determine the maximum x and y values on the screen. The point (0,0) is the left bottom of the screen, (maxx, maxy) is the right top, etc.

        View.Set ("graphics")
        % First draw a line from the left bottom to right top
        Draw.Line (0, 0, maxx, maxy, 1) 
        % Now draw a line from the left top to right bottom
        Draw.Line (0, maxy, maxx, 0, 1)
Execute  

Details   The screen must be in a "graphics" mode. See the View.Set procedure for details. If the screen is not in a "graphics" mode, an error will occur.

Status   Exported qualified.

This means that you can only call the function by calling Draw.Line, not by calling Line.

See also   View.Set, maxx, maxy and the various procedures in the Draw unit.
