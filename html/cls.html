clsclear screen graphics procedure

Syntax   cls

Description   The cls (clear screen) procedure is used to blank the output window. The cursor is set to the top left (to row 1, column 1).

Details   The entire output window is set to the current text background color (as set by colorback or Text.ColorBack).

The output window must be in "graphics" mode. See setscreen for details.

See also   See also predefined unit Text.
