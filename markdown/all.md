
#all

##Syntax
_setTypeName_ (**all**)


##Description
Given a set type named _S_, the set of all of the possible elements of _S_ is written _S_ (**all**).


##Example


        type smallSet : set of 0 .. 2
        var x : smallSet := smallSet (all)
                        % Set x contains elements 0, 1 and 2
