
#all

##Syntax
*setTypeName* (**all**)



##Description
Given a set type named *S*,* *the set of all of the possible elements of *S* is written *S *(**all**).



##Example



        type smallSet : set of 0 .. 2
        var x : smallSet := smallSet (all)
                        % Set x contains elements 0, 1 and 2
