
#abs

##Syntax
**abs** (_expn_)



##Description
The **abs** function is used to find the absolute value of a number (the _expn_). For example, **abs** (-23) is 23.



##Example
This program outputs 9.83.


        var x : real := -9.83
        put abs (x) % Outputs 9.83
