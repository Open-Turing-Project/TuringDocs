Str

Description

This unit contains the predefined constants for manipulating strings.

Several routines in the Str module are exported unqualified.

Entry Points

Lower   Convert a string to lower case.
Upper   Convert a string to upper case.
Trim   Remove leading and trailing blanks.
index   Finds a specified string in another string.
length   Returns the length of a string.
repeat   Creates a string by repeating a specified string a number of times.
