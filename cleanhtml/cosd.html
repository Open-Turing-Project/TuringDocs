cosd

Syntax

cosd (r : real) : real

Description

The cosd function is used to find the cosine of an angle given in degrees. For example, cosd (0) is 1.

Example

This program prints out the cosine of 30, 60, 90, up to 360 degrees.

        for i : 1 .. 12
            const angle := i * 30
            put "Cos of ", angle, " is ", cosd (angle)
        end for
